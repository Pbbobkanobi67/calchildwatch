# California Public Records Act (CPRA) Request Templates

Use these templates to request childcare subsidy payment data from California agencies.

---

## Template 1: California Department of Social Services

**Send to:** publicrecords@dss.ca.gov

**Subject:** CPRA Request - Childcare Subsidy Payment Data

---

Dear Public Records Coordinator,

Pursuant to the California Public Records Act (Government Code ยง7920.000 et seq.), I hereby request access to inspect and obtain copies of the following public records:

**Requested Records:**

1. **CalWORKs Stage 1, 2, and 3 Child Care Program Payment Data**
   - All payments disbursed to childcare providers in San Diego County
   - For fiscal years: 2022-2023, 2023-2024, and 2024-2025 (year-to-date)
   - Include: Provider name, facility license number, address, payment amount, payment date, program type

2. **Alternative Payment Program (APP) Data**
   - Total reimbursements by provider
   - Number of children served per provider (aggregate)

3. **Child Care and Development Fund (CCDF) Expenditures**
   - Payments to licensed centers and family childcare homes
   - By provider, for San Diego County

**Format Request:**
Please provide records in electronic format, preferably CSV, Excel (.xlsx), or database export. If records are maintained in a database, I request the data be exported with field names included.

**Fee Waiver Request:**
I request a fee waiver as this information is sought for a civic transparency project that will benefit the public interest. The data will be used for an open-source investigation into the allocation of childcare subsidies and will be made publicly available.

**Exemptions:**
If you determine that any portion of the requested records is exempt from disclosure, please:
1. Identify the specific exemption(s) claimed
2. Provide all non-exempt portions of the records
3. Provide a Vaughn index if multiple exemptions apply

**Response Timeline:**
Per Government Code ยง7922.535, I expect a response within 10 days indicating whether you will comply with this request.

Thank you for your attention to this matter.

## Tips for Effective CPRA Requests

1. **Be specific** - Name exact programs, date ranges, and data fields
2. **Request electronic format** - Easier to analyze
3. **Ask for fee waiver** - Cite public interest
4. **Know your rights** - 10 days for determination, judicial remedies available
5. **Follow up** - If no response, send reminder after 10 days
6. **Document everything** - Keep copies of all correspondence

## What to Do If Denied

1. Ask for written explanation citing specific exemptions
2. Request all non-exempt portions
3. Consider consulting with First Amendment Coalition or ACLU of California
4. File petition in Superior Court if unlawfully denied (Gov. Code ยง7923.000)
